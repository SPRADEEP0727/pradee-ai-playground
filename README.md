# Pradeep's AI Playground

This repository contains projects and code created as part of the **Evonuera Gen AI System Architect** program learnings.

## Projects

### 1. Flask Demo (`flaskdemo/`)
A simple Flask API with GET and POST endpoints for arithmetic operations (addition, subtraction, multiplication, division).

### 2. Snake Game (`game/`)
A classic snake game built with Python.

### 3. Python Basics (`python_basics/`)
Foundational Python scripts covering core concepts.

### 4. Streamlit Demo (`streamlitdemo/`)
A demo application built using Streamlit.

### 5. LangChain (`Langchain/`)
Experiments and examples using the LangChain framework.

### 6. RAG (`RAG/`)
Retrieval-Augmented Generation implementation.

### 7. Adaptive RAG (`adaptive_rag/`)
An adaptive RAG system that dynamically selects retrieval strategies based on query type.

### 8. Knowledge Graph RAG (`knowledge_graph_rag/`)
A RAG system that leverages knowledge graphs for structured information retrieval and generation.

### 9. KG RAG Notebook (`KG_RAG/`)
Jupyter notebook-based exploration of Knowledge Graph RAG techniques.

### 10. Graph RAG (`graph_rag/`)
Graph-based Retrieval-Augmented Generation experiments.
